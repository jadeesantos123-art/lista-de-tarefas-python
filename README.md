# lista-de-tarefas-python
Sistema simples de lista de tarefas desenvolvido em Python, permitindo adicionar, listar, concluir e remover tarefas, com foco em praticar lógica de programação e manipulação de listas e dicionários.


# Lista de Tarefas (To-Do List)

Projeto desenvolvido para a disciplina de Introdução à Programação.

## 📌 Funcionalidades
- Adicionar tarefas
- Listar tarefas
- Marcar tarefas como concluídas
- Remover tarefas

## 🛠️ Tecnologias
- Python

## 🎯 Objetivo
Praticar conceitos básicos como:
- Listas
- Dicionários
- Estruturas de decisão
- Laços de repetição

## ▶️ Como executar
Execute o arquivo `main.py` em um interpretador Python.
